Історія мови HTML. HTML (Hyper Text Markup Language – мова гіпертекстової розмітки) був створений на початку 1990 року працівником CERN (European Organization for Nuclear Research) Тімом Бернерс-Лі. Мова html була заснована на SGML (Standard Generalized Markup Language – стандартна узагальнена мова розмітки). Основною ідеєю SGML було створення між платформенної мови для розмітки документів. Цю ідею реалізувала команда під керівництвом Чарльза Гольдфарба з компанії IBM.
